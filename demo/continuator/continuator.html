Continuator demo

Continuator continues a moleculizer simulation from the point where state was dumped from the moleculizer simulation. When a moleculizer simulation has a clock-time limit and it exceeds the limit, then it automatically dumps state into a file called timeout-state.xml. Continuator will continue the simulation from the time-out point in this case, or from the point that state was dumped from a moleculizer simulation by the user with a "dump-state" event.

In this demo, a moleculizer simulation (a version of the alpha pathway receptor complex) runs until it times out, then continuator continues the simulation, again with a time limit just to make the demo brief.
