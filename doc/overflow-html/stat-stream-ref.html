stat-stream-ref/@name

Here are the simulator statistics that you can dump:

sim-time
The number of simulated seconds since the start of the simulation.
clock-time
The number of actual seconds since the start of the simulation.
species-count
The number of species.
reaction-count
The number of reactions.
reaction-event-count
The number of reaction events that have occured since the start of the simulation. For example, each dimerization of two molecules is a reaction event.
volume
The volume in liters.
